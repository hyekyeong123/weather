#날씨알리미 앱
##open api를 활용한 날씨앱
----------------------------------------------------------------------------------------------------
###기능
>일주일단위로 매일 아침의 날씨를 확인할 수 있게 하였습니다.
>서울, 대전, 대구, 부산, 런던, 뉴욕의 날씨를 확인할 수 있게 하였습니다.
>날씨에 따라서 사진도 변경하게 만들었습니다. 
>open waether web을 사용하였습니다. 

###사용한 언어
>HTML, CSS, Jquery, Ajax
